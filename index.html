New Games app

Create folders for icons

How it currently works

  1. Codes get stored in JSON, duplicates get cleaned up
  2. Create folders -> creates folders, based on stored game codes in JSON
  3. Create game/sym links -> creates Icons.txt file based on stored game codes

This seperate/modular way still allows just creating folders, later game/sym link generation should be toggled based on inputs or JSON codes
